# goods-shop

굿즈 판매 테스트 쇼핑몰의 **공개 화면 코드**입니다. (HTML / CSS / JavaScript)

라이브 사이트: https://9yurii.github.io/goods-shop/

## 알아두실 점

- 결제는 **토스페이먼츠 테스트 모드**로 동작합니다. 실제 카드 청구가 발생하지 않습니다.
- 빌드 도구가 없습니다. 브라우저가 파일을 그대로 읽습니다.
- 백엔드(데이터베이스, 로그인, 결제 승인)는 Supabase 에 있습니다.

## 로컬에서 보기

파일을 더블클릭해서 여는 방식(`file://`)으로는 작동하지 않습니다. 로컬 웹 서버가 필요합니다.

```bash
npx serve .
```

## 구조

```
index.html            상품 목록 (홈)
pages/                로그인 / 회원가입 / 결제 / 내 결제 내역 / 관리자
js/                   페이지별 로직
assets/css/style.css  스타일 (모바일 우선 반응형)
```

`js/config.js` 에 들어있는 값(Supabase anon 키, 토스 클라이언트 키)은 **원래 브라우저에 공개되는 값**입니다. 비밀키가 아닙니다.
